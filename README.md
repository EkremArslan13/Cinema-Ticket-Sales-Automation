# 🎬 Sinema Bilet Satış Otomasyonu

Bu proje, C# programlama dili ve Windows Forms arayüzü kullanılarak geliştirilmiş, dinamik bir bilet satış yönetim sistemidir. 

## 🚀 Proje Öne Çıkanlar
* **Taşınabilirlik (Portability):** Proje, `Relative Path` (Göreceli Yol) mantığıyla kurgulanmıştır. Sabit dosya yolları yerine `Application.StartupPath` kullanılarak, projenin her bilgisayarda hatasız çalışması sağlanmıştır.
* **Dinamik Varlık Yönetimi:** Resimler ve kaynak dosyaları, derleme sırasında otomatik olarak çıktı dizinine kopyalanacak şekilde konfigüre edilmiştir.

## 🛠️ Kullanılan Teknolojiler
* **Dil:** C#
* **Platform:** .NET 8.0 (veya senin kullandığın sürüm)
* **Arayüz:** Windows Forms (WinForms)
* **Versiyon Kontrol:** Git & GitHub

## 📂 Kurulum ve Çalıştırma
1. Bu depoyu (repository) bilgisayarınıza klonlayın: 
   `git clone https://github.com/EkremArslan13/Cinema-Ticket-Sales-Automation.git`
2. `CinemaTicketsSellOtomation.sln` dosyasını Visual Studio ile açın.
3. Bağımlılıkların yüklenmesi için projeyi **Build** edin.
4. **F5** tuşuna basarak uygulamayı başlatın.
